# 더치페이 계산기 3.0 🍃

**토스/애플 스타일의 미니멀 디자인**을 적용한 고급 웹 기반 더치페이 계산기입니다. 일본 여행이나 모임에서 간편하게 비용을 분담하고 PayPay로 바로 송금할 수 있습니다.

## ✨ 주요 특징

### 🎨 **미니멀 디자인**
- **토스/애플 디자인 철학** 기반의 깔끔한 UI
- **다크모드 자동 전환** (시스템 설정 연동)
- **불필요한 여백 제거**로 컴팩트한 레이아웃
- **직관적인 버튼 구성**과 현대적 감각

### 💰 **PayPay 연동**
- **PayPay 앱 바로 실행** (모바일에서 `paypay://` 스킴 사용)
- **PC에서는 PayPay 웹사이트 자동 이동**
- **정산 완료 후 원터치 송금**

### 👥 **고급 사용자 관리**
- **프로필 전용 관리 페이지** (`profiles.html`)
- **메인 화면 드롭다운**으로 빠른 프로필 전환
- **프로필별 독립 데이터** 및 삭제 기능

### 📂 **스마트 카테고리 시스템**
- **용도별 내역 분류** (일본여행, 생일잔치 등)
- **카테고리별 독립 관리**
- **실시간 전환** 및 초기화 기능

## 📱 사용 시나리오

### 시나리오 1: 일본 여행 정산
```
1. 프로필 설정 → "철수" 프로필 생성
2. 카테고리 → "도쿄여행" 새 카테고리 생성
3. 참가자 추가 → 철수, 영희, 민수 추가
4. 지출 입력:
   - 호텔비: ¥15,000 (철수 지불)
   - 저녁식사: ¥8,000 (영희 지불)
   - 교통비: ¥3,000 (민수 지불)
5. 정산 확인 → 자동 계산된 결과 확인
6. PayPay 송금 → "PayPay로 보내기" 버튼으로 즉시 송금
```

### 시나리오 2: 다중 모임 관리
```
- 기본 카테고리: 일상 모임
- 회사 회식 카테고리: 업무 관련 지출
- 생일잔치 카테고리: 특별 이벤트
- 각 카테고리별 독립 관리 및 정산
```

## 📲 PayPay 연동 상세

### **모바일 (Android/iOS)**
```javascript
// PayPay 앱 스킴으로 직접 실행
window.location.href = 'paypay://';

// 앱이 없으면 1.5초 후 웹사이트로 이동
setTimeout(() => {
    if (!document.hidden) {
        window.open('https://paypay.ne.jp/', '_blank');
    }
}, 1500);
```

### **PC (Desktop)**
```javascript
// 바로 PayPay 웹사이트로 이동
window.open('https://paypay.ne.jp/', '_blank');
```
