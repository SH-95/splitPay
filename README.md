# 더치페이 계산기

간단하고 직관적인 웹 기반 더치페이 계산 웹사이트입니다. 여러 명이 함께 식사하거나 활동할 때 비용을 공정하게 분담하고 정산할 수 있습니다.

## 🚀 주요 기능

### ✅ 완성된 기능

1. **참가자 관리**
   - 참가자 이름 입력 및 추가
   - 참가자 목록 표시 (태그 형태)
   - 참가자 개별 삭제 기능

2. **지출 내역 관리**
   - 항목명, 금액, 지불자 입력
   - To-do 리스트 형태의 지출 내역 표시
   - 지출 내역 개별 삭제 기능
   - 실시간 총 금액 자동 계산

3. **실시간 계산**
   - 총 지출 금액 자동 계산
   - 참가자 수 자동 카운트
   - 1인당 부담 금액 실시간 계산

4. **정산 결과**
   - 누가 누구에게 얼마를 줘야 하는지 자동 계산
   - 최소한의 거래로 정산 완료하는 알고리즘
   - 직관적인 정산 결과 표시

5. **사용자 경험**
   - 미니멀하고 직관적인 UI 디자인
   - 반응형 웹 디자인 (모바일 최적화)
   - 엔터키 입력 지원
   - 실시간 데이터 업데이트

## 🎯 기능별 진입점 (URI)

- **메인 페이지**: `index.html`
- **스타일시트**: `css/style.css`
- **JavaScript 로직**: `js/main.js`

### 주요 JavaScript 함수

- `addParticipant()` - 참가자 추가
- `removeParticipant(name)` - 참가자 제거
- `addExpense()` - 지출 내역 추가
- `removeExpense(id)` - 지출 내역 제거
- `updateDisplay()` - 전체 화면 업데이트
- `updateSettlement()` - 정산 결과 계산 및 표시

## 💡 사용 방법

1. **참가자 등록**: 참가자 이름을 입력하고 "추가" 버튼을 클릭
2. **지출 입력**: 항목명, 금액, 지불한 사람을 선택하고 "지출 추가" 버튼을 클릭
3. **결과 확인**: 실시간으로 계산되는 총액, 1인당 부담금, 정산 결과를 확인
4. **정산 완료**: 정산 결과에 따라 실제 송금 진행

- **태블릿**: 중간 화면 크기 최적화

## 🧮 정산 알고리즘

효율적인 정산을 위해 다음 알고리즘을 사용합니다:

1. 각 참가자의 지불 금액 계산
2. 1인당 부담금과의 차액으로 채무/채권 관계 파악
3. 채무자와 채권자를 매칭하여 최소 거래 횟수로 정산
4. 소수점 오차 처리로 정확한 계산
