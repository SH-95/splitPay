# 더치페이 계산기 4.0 🌍

**다국어 지원** 및 **고급 관리 기능**을 갖춘 완전한 더치페이 솔루션입니다. 일본 여행이나 국제적 모임에서 PayPay로 바로 송금할 수 있습니다.

## ✨ 4.0 버전 주요 업데이트

### 🌏 **완벽한 다국어 지원**
- **한국어 🇰🇷 / 일본어 🇯🇵** 실시간 전환
- **모든 UI 텍스트** 완벽 번역 (버튼, 알림, 오류 메시지 등)
- **언어 설정 자동 저장** LocalStorage 기반

### 🔧 **고급 관리 기능**
- **카테고리 수정/삭제** - 인라인 편집으로 간편하게
- **프로필 수정/삭제** - 클릭 한번으로 이름 변경
- **고유 ID 시스템** - 동일 이름 프로필도 별도 관리

### 📱 **PayPay 앱 연동 개선**
- **iframe 방식** 앱 실행으로 안정성 향상
- **페이지 숨김 감지**로 정확한 앱 실행 판별
- **PC/모바일 환경별** 최적화된 처리

### 🐛 **주요 버그 수정**
- **확인 메시지 줄바꿈** 정상 처리 (`\\n` → 실제 줄바꿈)

- **카테고리 추가 오류** 수정

## 🔧 고급 관리 기능

### **1. 카테고리 관리**
```
✏️ 수정: 카테고리 선택 → 편집 아이콘 → 이름 입력 → 저장
🗑️ 삭제: 카테고리 선택 → 삭제 아이콘 → 확인
➕ 추가: "+ 새 카테고리" → 이름 입력 → 추가
```

### **2. 프로필 관리**
```
👤 추가: profiles.html → 이름 입력 → 추가
✏️ 수정: 프로필 카드 → 편집 아이콘 → 이름 수정 → 저장
🗑️ 삭제: 프로필 카드 → 삭제 아이콘 → 확인
🔄 전환: 메인 화면 드롭다운 또는 프로필 관리에서 "선택"
```

### **3. 고유 ID 시스템**
```json
{
  "profiles": {
    "guest_001": { "name": "guest", "id": "guest_001" },
    "john_001": { "name": "John", "id": "john_001" },
    "john_002": { "name": "John", "id": "john_002" }  // 같은 이름도 별도 관리
  }
}
```

## 📱 PayPay 연동 상세

### **모바일 환경**
```javascript
// 1. iframe으로 앱 실행 시도
const iframe = document.createElement('iframe');
iframe.src = 'paypay://';

// 2. 1초 후 웹사이트로 이동 준비
setTimeout(() => {
    window.open('https://paypay.ne.jp/', '_blank');
}, 1000);

// 3. 앱 실행 시 (페이지 숨김) 웹사이트 이동 취소
document.addEventListener('visibilitychange', handleAppLaunch);
```

### **PC 환경**
```javascript
// 바로 PayPay 웹사이트로 이동
window.open('https://paypay.ne.jp/', '_blank');
```
