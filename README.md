# 💸 더치페이 계산기 5.0

한국어/일본어를 지원하는 모바일 친화적 더치페이 정산 앱입니다.
누가 얼마를 냈는지 입력하면 **최소 거래 수**로 정산 결과를 계산해줍니다.

---

## 🚀 실행 방법

```bash
# Node.js 설치 필요 (v14+)
cd splitPay
npx serve . -p 3000
```

브라우저에서 **http://localhost:3000** 접속

---

## ✨ 주요 기능

### 탭 구조 (하단 탭바)
| 탭 | 기능 |
|---|---|
| 👥 참가자 | 카테고리 관리 + 참가자 추가/삭제 |
| 💰 지출 | 지출 입력/수정/삭제 |
| 📊 정산 | 총액 요약 + 정산 결과 |

### 핵심 기능
- **최소 거래 정산** — 가장 적은 송금 횟수로 계산
- **다중 프로필** — 여러 사람이 각자 데이터 분리 관리
- **카테고리** — 여행별, 모임별로 독립 관리
- **PayPay 연동** — 정산 후 바로 PayPay 앱 실행
- **완전 오프라인** — 모든 데이터 브라우저 LocalStorage 저장
- **다국어** — 한국어 🇰🇷 / 일본어 🇯🇵 실시간 전환
- **다크모드** — 시스템 설정 자동 감지

## 📱 사용 방법

### 1단계 — 참가자 등록
1. **👥 참가자** 탭 이동
2. 이름 입력 후 `+` 버튼 (또는 Enter)
3. 참가자 칩이 추가됨

### 2단계 — 지출 입력
1. **💰 지출** 탭 이동
2. 항목명 / 금액 / 지불한 사람 입력
3. `지출 추가` 버튼 클릭
4. 카드 클릭 시 수정, × 버튼으로 삭제

### 3단계 — 정산 확인
1. **📊 정산** 탭 이동
2. 총액 / 인원 / 1인당 금액 확인
3. `A → B ¥1,000` 형태로 누가 누구에게 얼마를 보낼지 표시
4. PayPay로 바로 송금 가능
